# Cecil theme installer

The [Composer](https://getcomposer.org) installer for [Cecil](https://cecil.app)'s themes.

## Usage

Add the following lines to the `composer.json` file of your theme:

```json
"type": "cecil-theme",
```

```json
"require": {
  "cecil/theme-installer": "^2.0"
}
```

```json
"extra": {
  "name": "<theme_name>",
}
```
