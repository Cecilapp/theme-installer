# Cecil theme installer

> [Composer](https://getcomposer.org) installer for [Cecil](https://cecil.app)'s themes.

## Usage

Add the following lines the `composer.json` file of your theme:

```json
"type": "cecil-theme",
```

```json
"require": {
    "cecil/theme-installer": "^1.4"
}
```

```json
"extra": {
    "name": "THEME_NAME",
}
```
