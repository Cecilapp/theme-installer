# PHPoole theme installer

[Composer](https://getcomposer.org) installer for [PHPoole](https://phpoole.org) themes.
 
## Usage

Add the following to your theme `composer.json`:

```json
"type": "phpoole-theme",
```

```json
"require": {
    "phpoole/theme-installer": "^1.1"
}
```

```json
"extra": {
    "name": "THEME_NAME",
}
```
