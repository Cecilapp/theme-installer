# PHPoole-installer

Composer installer for PHPoole themes and plugins.

Package type supported:
* phpoole-theme 
* phpoole-plugin
 
Installation
------------

Add the following to your PHPoole theme or plugin composer.json:

```json
"require": {
    "narno/phpoole-installer": "1.0.x-dev"
}
```

Name
----

```json
"extra": {
    "name": "theme-or-plugin-name",
}
```
